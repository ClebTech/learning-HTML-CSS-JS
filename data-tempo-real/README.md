# Relógio Digital em Tempo Real

Um simples relógio feito com **HTML**, **CSS** e **JavaScript**.

## Funcionalidades
- Mostrar o dia da semana, a data e a hora atual.
- Atualização em tempo real.
- Responsivo para diferentes tamanhos de tela.
